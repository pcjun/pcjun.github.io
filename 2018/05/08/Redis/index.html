Redis

Redis支持的类型

  • 字符串 String
  • 列表 List
  • 哈系 Hash
  • 集合 Set
  • 有序集合 zset或者sorted set

  • 下面是简单的一些操作

    • String

    • Hash

    • List

    • Set

    • SortedSet

Redis 键 (key) — 用于管理redis的键

  • 设置：set key value
  • 设置过期时间：expire key seconds
    • time为秒
  • 检查key是否存在：exists key
  • 获取：get key
  • 删除：del key
  • 设置过期时间以毫秒计算：pexpireat key millseconds
    • millseconds为毫秒
  • 将key重命名：rename key newkey
  • 当newkey不存在时，以newkey做为新的键：renamenx key newkey

    Redis 字符串命令

  • 返回key中字符串的子字符串：getrange key start end
    • start:从 0 开始
    • end：结束下标
  • getset设置旧值到新的key
  • 设置一个或者多个key-value：msetnx k1 v1 k2 v2 k3 v3
  • 获取一个或者多个key的值：mget k1 k2 k3
  • 获取设置key的长度：strlen key
  • 追加字符串：append key value

    小结

  • 列表的操作：lpush lrange
  • String的操作：set get
  • Hash: hmset hget
  • Set: sadd key value smembers key
